# Reno Source Financial Projections Dashboard — V4.0

Interactive 24-month financial model with M1 + M2 actual results.

**Live:** renosource-projections-v4.netlify.app  
**For accredited investors only.**

## Features
- M1 (April 2026) and M2 (May 2026) actual results highlighted in orange
- M1 Proj / M2 Proj columns show original pre-launch projections for comparison
- Interactive sliders to stress-test M3-M24 forward projections
- Revenue and EBITDA bar charts
- Full 24-month P&L table
- Single-file HTML — no build step required
