# Reno Source Financial Projections V4.0

**24-Month Pro Forma Model — M1 + M2 Actuals**

A fully self-contained, interactive financial model for Reno Source, reflecting two months of actual operating results (April & May 2026) and 22 months of forward-looking projections.

---

## 📊 What's Inside

- **Two Months of Actuals**: M1 (April 2026) and M2 (May 2026) real performance data
- **22-Month Forward Projections**: M3–M24 updated based on observed trends
- **Interactive Sliders**: Stress-test assumptions (AOV, gross margin, CPC sensitivity)
- **Live Charts**: Revenue and EBITDA visualizations that update instantly
- **Full P&L**: Month-by-month breakdown with actuals vs. projections highlighted
- **Unit Economics**: Conversion rate, AOV, CAC, and CPC evolution

---

## 🚀 Quick Start

1. **Open `index.html` in any modern browser** (Chrome, Safari, Firefox, Edge)
2. No installation needed — everything is self-contained
3. Scroll through the dashboard to see KPIs, charts, and the full P&L
4. Click "Stress Test the Model" to adjust assumptions with sliders

## 🧪 Stress Testing

Use the interactive sliders to model different scenarios:

1. **AOV Multiplier** (±20%): Bigger or smaller customer baskets?
2. **Gross Margin** (±5%): Supplier terms or freight cost changes?
3. **CPC Sensitivity** (±25%): Cheaper or more expensive clicks?

**Note**: Sliders only affect M3–M24. M1 and M2 actuals stay fixed.
